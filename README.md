# Portfolio Builder Web App
A full-stack web application that allows users to create, 
customise, and share their developer portfolio with a unique link.

## Live Demo
https://portfolio-vkm9.onrender.com

## Features
Create personalized developer portfolios
Unique username-based portfolio URLs
Add multiple projects dynamically
Github & Linkedin integration
Download portfolio as PDF
Clean and responsive UI
MongoDB database integration

## Tech Stack
Frontend
EJS(Embedded javaScript Templates)
HTML5, CSS3

Backend
Node.js
Express.js

Database
MongoDB(Mongoose)

## Install Dependencies
npm install

## Run the Project
npm start
node app.js
Open: http://localhost:3000
